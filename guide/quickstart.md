# Quickstart

ByteRover Team Memory is a hosted remote MCP for OpenAI Codex memory MCP.

## Fast Path

1. Open ByteRover Team Memory and select the buyer plan.
2. Create or request a bearer token from the hosted product.
3. Add https://byterovermemory.clauxel.com/mcp to a compatible MCP client.
4. Run tools/list, then call memory_recall with public-safe sample data.
5. Save the returned receipt or export for human review.
